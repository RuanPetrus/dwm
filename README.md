# Dwm
This is my version of dwm using [dwm-flexipatch](https://github.com/bakkeby/dwm-flexipatch).

# Patchs
- Status 2D for colors in status bar
- Systray patch
- Underline Tags
- Status Padding
- Self restart
- Stacker for better control
- Toogle Full Screen
- Vanity Gaps
- Swallow
